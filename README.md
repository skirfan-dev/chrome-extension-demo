# Chrome Extension Demo

> A Chrome Extension built using Manifest V3 demonstrating modern extension architecture.

## Features

- Manifest V3
- Popup Interface
- Content Scripts
- Runtime Messaging
- Chrome Storage API
- Modern UI
- DOM Interaction

## Technologies

- JavaScript
- HTML5
- CSS3
- Chrome Extension API

## Installation

1. Clone this repository.
2. Open Chrome.
3. Navigate to `chrome://extensions`.
4. Enable Developer Mode.
5. Click **Load unpacked**.
6. Select this project folder.

## Project Structure

```
manifest.json
content.js
injected.js
popup.html
popup.js
icon.png
```

## Learning Goals

This project demonstrates:
